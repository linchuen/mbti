# formation-process-guide.md

## 頁面目的

此頁面補上「如何判斷人格形成流程」的教學。

重點：

* 教使用者觀察自己的認知處理順序
* 以流程判斷，不以人格特質詞判斷
* 產出候選認知功能路徑，不直接下類型結論

不得做人格測驗結果頁，不得直接判定使用者是某一類型。

## 1. 頁面引導

固定文案：

> 先判斷流程，再理解人格

> 你要找的不是「我像誰」，而是「我如何感知、判斷並形成穩定偏好」

---

## 2. 榮格的兩大方向

需明確說明：榮格認為人類理解世界有兩大方向。

| 方向 | 意思 |
| ---- | ---- |
| 外向 (Extraverted) | 從外界獲取資訊、與現實互動 |
| 內向 (Introverted) | 從內心模型理解世界 |

---

## 3. 四種處理方式

需明確說明：在方向之外，人會透過四種功能處理資訊與判斷。

| 類型 | 功能 | 本質 |
| ---- | ---- | ---- |
| 感知 | Sensing (S) | 看見現實 |
| 直覺 | Intuition (N) | 看見可能 |
| 思考 | Thinking (T) | 判斷正確 |
| 情感 | Feeling (F) | 判斷價值 |

---

## 4. 判斷前提（先看認知，不看標籤）

必須包含以下三條原則：

* 以處理資訊方式為判斷對象
* 以長期重複模式為判斷證據
* 以壓力下的反應偏移做交叉驗證

---

## 5. 人格形成流程判斷（四步）

固定主流程文字（不可修改）：

感知方式 → 判斷方式 → 功能排序 → 人格傾向

頁面需有明確「箭頭導引」讓讀者知道下一步。

### Step 1 感知方式

判斷使用者習慣先抓取：

* 外部即時訊號（Se / Ne 傾向）
* 內部既有脈絡（Si / Ni 傾向）

### Step 2 判斷方式

判斷使用者決策時優先：

* 邏輯一致性（Ti / Te 傾向）
* 價值一致性（Fi / Fe 傾向）

### Step 3 功能排序

依自然使用頻率 + 壓力時失衡點推估：

* 優勢功能（dominant）
* 輔助功能（auxiliary）
* 第三功能（tertiary）
* 劣勢功能（inferior）

### Step 4 人格傾向

輸出為認知運作傾向摘要，不得輸出最終類型判定。

---

## 6. 判斷紀錄方式

至少要求 3 筆觀察紀錄，每筆需包含：

* 情境描述（做了什麼決策）
* 先感知到什麼
* 最後用什麼標準下判斷
* 事後壓力反應

判斷依據必須來自多次紀錄，不得用單一事件下結論。

---

## 7. 常見誤判修正

至少包含以下三項：

* 我外向，所以一定是 Fe/Se → 回到判斷標準，不看社交表現
* 我重感受，所以一定是 Fi → 區分個人價值判斷與情緒強度
* 我工作理性，所以一定是 T 主導 → 分辨角色要求與自然偏好
